Passo-a-passo provisionamento Kayrós Link

Este documento tem como intuito auxiliar os colaboradores do suporte técnico a realizar um provisionamento.

Primeiramente para realizar um provisionamento é necessário aguardar que o técnico encaminhe 3 fotos no grupo de provisionamento, entre elas:

  • Foto do sinal interno. (residência)
  • Foto do sinal externo. (CTO)
  • Foto do serial da ONU.

Um segundo ponto importante é ter ciência das normas para que um provisionamento seja aceito. Você deve se basear sempre pelo sinal externo (CTO), ao provisionar e verificar o sinal no UNM é necessário verificar se está dentro do padrão aceitável de 1,5dbm. Um exemplo:

O técnico encaminhou uma foto com o sinal da CTO em -20dbm, ao fazer o cálculo você vai somar o valor aceitável com o valor da caixa, ou seja: -20dbm + -1,5dbm = -21,5dbm. Caso o sinal da ONU ultrapasse, não será aceitado. No caso: 21,70dbm,21,80dbm e em sequência não seriam aceitados, porém, qualquer valor abaixo seria aceitado.

Para realizar o provisionamento primeiro é necessário consultar o ID do cliente informado pelo técnico → ir em serviços → habilitar o serviço → copiar o nome completo do cliente → ir até a opção rede → equipamentos e conexão → selecionar a OLT correspondente com o endereço do cliente → ações → ver clientes conectados → Pendentes camada 2 → atualize → verifique o serial da ONU.

Após consultar o serial da ONU pela foto encaminhada pelo técnico e caso seja a mesma que listou, realize o seguinte procedimento: clique em ações → autorizar → selecione o modelo da ONU conforme a foto (os modelos utilizados são 01A MINI, 02B BRIDGE, 02FG) → cole o nome completo do cliente → selecione o serviço (ao selecionar o serviço, automaticamente o login e senha do cliente será preenchido) → digite a VLAN da determinada OLT onde está sendo feito o provisionamento → clique em autorizar CPE → após alguns minutos o provisionamento será concluído.

O provisionamento terá sido realizado, para concluir o último passo será verificar o sinal. Para consultar o sinal deve ser realizado o seguinte procedimento: Copie o serial da ONU → Abra o UNM → Selecione a OLT no qual aquela ONU foi provisionada → selecione a opção “ONU list” → cole o serial da ONU → selecione a respectiva ONU → clique com o botão direito → selecione a opção “Service Configuration” → Após isso, selecione a opção na barra lateral chamada de “Opt Module Para Information” → a informação do sinal estará mostrando da seguinte forma: Recive power dBm.

Caso o sinal esteja de forma satisfatória, ou seja, dentro do padrão de 1,5 dBm de diferença, deve ser encaminhado a foto do sinal no grupo, respondendo o ID do cliente que o técnico encaminhar e informar: Sinal ok. Provisionado por ...... (nome de quem provisionou)

Mas e se caso o sinal ultrapasse a marca de 1,5 dBm? O que deve ser feito?

Neste caso é necessário ser realizado um procedimento de 3 etapas a ser seguido com o propósito de normalizar o problema.

  1. O primeiro passo vai ser solicitar que o técnico realize uma limpeza no equipamento.
  2. Caso após a limpeza do equipamento o sinal não tenha normalizado, solicite ao técnico que refaça os conectores
  3. E se caso o passo anterior não tenha normalizado o sinal, solicite para o técnico trocar a ONU.

Termos técnicos

Ao longo deste documento você deve ter notado alguns termos técnicos utilizados para se referir aos equipamentos e sistemas de fibra óptica, nessa parte será explicado alguns termos informados anteriormente para facilitar o seu entendimento sobre o assunto.

  • OLT: (Optical Line Terminal) é o equipamento responsável por gerir e distribuir o acesso aos serviços de dados e streaming. Se você ainda tem dúvidas sobre o que é a OLT, vamos entender algumas outras nomenclaturas que envolvem o fornecimento de rede de fibra óptica e que podem ajudar nessa compreensão.

    Uma delas é o padrão FTTh – fibra até a residência (em português) fornecido através do sistema PON (Redes Ópticas Passivas). Entre os equipamentos e tecnologias básicas que compõem sua arquitetura está a OLT.

    A OLT irá centralizar e distribuir os sinais ópticos sobre fibra óptica até a chegada ao cliente, no caso uma ONU (Unidade de Rede Óptica) ou ONT (Terminal de Rede Óptica).

    Enquanto isso, para garantir o controle e fluxo de dados para a ONU ou ONT está a OLT, que é normalmente instalada na central operadora. A OLT, portanto, é o terminal de linha óptica responsável por converter e transmitir os dados para a rede PON. Em outras palavras, a OLT é um equipamento que fica instalado no provedor com a função de fazer a distribuição do serviço de dados, streaming, voz, imagem e telefonia IP. Além de enviar os dados, a OLT também é responsável por receber informações de uma rede de longa distância para redistribuição.

  • VLAN

    A Virtual Local Area Network (VLAN) é uma rede independente que visa se conectar e se comunicar a redes físicas através de domínios de broadcast. É uma forma de separação lógica para melhorar a administração, o desempenho e a segurança.

    Atualmente a Kayrós Link possui 4 OLT ́s, estas são: OLT SOF, OLT ESTÂNCIA, OLT VALE e OLT MARAJÓ. Cada OLT possui sua VLAN na qual está vinculada a cada OLT. Se durante o provisionamento for inserido o número da VLAN incorreto a ONU irá “provisionar” porém não irá subir rede, devido a VLAN incorreta. A baixo estará o número correspondente a cada VLAN para que não tenha erro durante o provisionamento.

  • OLT SOF: 1001
  • OLT VALE: 1101
  • OLT ESTANCIA: 1007
  • OLT MARAJÓ: 1024
  • OLT JARDIM: 1028

  • ONU

    ONU é uma tecnologia cuja sigla significa Optical Network Unit e é uma rede de fibra óptica que possui a capacidade de converter um sinal OLT em sinal elétrico, distribuindo para outros canais de transmissão do serviço.

    Atualmente a Kayrós Link possui alguns modelos padrões de ONU ́s que são utilizadas, dentre estes modelos se destacam os seguintes:01-A MINI, 02B-BRIDGE, 02-FG e 02-B ROUTER. O modelo de cada ONU citada tem o seu propósito, entre eles:

    01-A MINI: Este é o modelo padrão que será utilizado em grande maioria dos provisionamentos.

    02B-BRIDGE: Este modelo é responsável por ser utilizado em ativações onde o cliente vai aderir telefonia, o que difere este modelo 02B do modelo 01-A é que este modelo possui uma entrada específica para telefonia.

    02-FG: Este modelo é utilizado de forma ROUTER, ou seja, este modelo atua como uma ONU e roteador em um só. Este modelo não é utilizado nos dias atuais, porém pode haver algum cliente com um plano antigo que utilize este modelo.
